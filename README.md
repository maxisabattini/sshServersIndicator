Ssh Servers Indicator
===================

AppIndicator for ssh servers connections, is for automatize login to ssh servers.

##Usage

Put config files in app servers folder.

Below is sample of config file:

```
# config file
title=Test Server
host=somehost.com
user=someuser
pass=thepassword
```
